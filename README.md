## Features

- **Multi-tab workspace** — AvalonDock 기반 도킹 레이아웃
- **Training** — 모델 학습 설정 및 실시간 진행률 모니터링
- **Test / Inference** — 단일 이미지 및 배치 예측
- **Model Architecture Builder** — 드래그 앤 드롭 신경망 설계 + PyTorch 코드 생성
- **Dashboard** — 학습 이력 및 시스템 통계
- **Model Manager** — 등록된 모델 관리
- **Dark theme** (VS Code style)

## Requirements

- Windows 10/11 (x64)
- [.NET 8 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)

## Installation

1. 위 테이블에서 최신 버전 zip 다운로드
2. 압축 해제
3. `AIStudio.WPF.exe` 실행
